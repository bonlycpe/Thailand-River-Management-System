# Thailand River Management System

Web application for Thailand River Management System using PHP and MVC in Database Systems Management class 

<img src="https://user-images.githubusercontent.com/106681968/215102189-883beea4-114e-4f41-aeb4-60f80fdcb78b.jpg" width=70% height=70%>

This web application can provide users to manage river information by themself and can see the summary of the river that they at once

Objective of this project
1. To study MVC software design and SQL 
2. To make a website that can help user to manage river information
